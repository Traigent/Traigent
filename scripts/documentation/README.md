# 📚 Documentation Scripts

Scripts for managing documentation, generating dashboards, and syncing docs.

## Scripts

- **`doc_dashboard.py`** - Interactive documentation health dashboard
- **`sync_docs.py`** - Documentation synchronization and management

## Usage

```bash
# Generate documentation dashboard
python scripts/documentation/doc_dashboard.py

# Simple text output
python scripts/documentation/doc_dashboard.py --simple

# JSON output for automation
python scripts/documentation/doc_dashboard.py --json

# Sync documentation
python scripts/documentation/sync_docs.py
```

## Features

- Documentation health metrics
- Coverage analysis
- Freshness tracking
- Quality assessment
- Consistency checking
