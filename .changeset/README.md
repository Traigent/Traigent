# Changesets

This repo uses [Changesets](https://github.com/changesets/changesets) for manual release
versioning and changelog generation.

- Add a changeset for any public API, export, or behavior change.
- Use `npm run changeset` to create a new entry.
- Use `npm run changeset:version` to apply pending version and changelog updates.

See [../docs/RELEASING.md](../docs/RELEASING.md) for the full release flow.
