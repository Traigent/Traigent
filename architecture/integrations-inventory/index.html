    Traigent Integrations Directory - Comprehensive Inventory¶

    Overview¶

    The traigent/integrations/ directory contains the framework integration system that enables zero-code optimization across multiple LLM providers and frameworks. This system uses a hybrid plugin + framework override architecture.

    Total Files: 66 Python files, regenerated from the current traigent/integrations/ tree. Total Integrations: Built-in coverage spans OpenAI, Anthropic, LangChain, Bedrock, Azure OpenAI, Cohere, Gemini, HuggingFace, LiteLLM, LlamaIndex, Mistral, PydanticAI, vector stores, and observability integrations.
