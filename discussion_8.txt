## RAG Example Upgrade

### Branch: `rag-up-wip`

Upgraded `examples/quickstart/02_customer_support_rag_real_llm.py` from fake keyword-matching RAG to real semantic RAG.

### Before (Keyword Matching)
```python
KNOWLEDGE_BASE = ["Returns accepted within 30 days...", ...]

def simple_retriever(query, k):
    # Naive keyword matching - NOT semantic
    score = sum(1 for word in query.split() if word in doc.lower())
```

### After (Real FAISS RAG)
```python
from langchain_community.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings
from langchain_text_splitters import RecursiveCharacterTextSplitter

def create_vector_store(chunk_size, chunk_overlap):
    embeddings = OpenAIEmbeddings(model="text-embedding-3-small")
    documents = chunk_documents(KNOWLEDGE_BASE_DOCUMENTS, chunk_size, chunk_overlap)
    return FAISS.from_documents(documents, embeddings)
```

### Key Features Added

1. **Long-form Knowledge Base**
   - 7 documents (~1,500-2,000 chars each)
   - ~10,000 total characters
   - Topics: returns, shipping, payments, support, hours, gift cards, pricing

2. **Document Chunking**
   - `RecursiveCharacterTextSplitter` from LangChain
   - Configurable `chunk_size` (300, 500, 800)
   - Configurable `chunk_overlap` (25, 50, 100) - sliding window
   - Splits on: paragraphs -> sentences -> words

3. **FAISS Vector Store**
   - OpenAI `text-embedding-3-small` embeddings
   - Semantic similarity search
   - Cached by `(chunk_size, chunk_overlap)` tuple

4. **Configuration Space**
   ```python
   CONFIGURATION_SPACE = {
       "model": ["gpt-3.5-turbo", "gpt-4o-mini", "gpt-4o"],
       "temperature": [0.1, 0.3, 0.5, 0.7],
       "k": [2, 3, 5],
       "chunk_size": [300, 500, 800],
       "chunk_overlap": [25, 50, 100],
   }
   ```

5. **Constraints**
   - GPT-4o: temperature <= 0.3
   - GPT-3.5-turbo: k >= 3
   - chunk_overlap < chunk_size (prevents invalid combos)

6. **Summary Output**
   - Accuracy by Model
   - Accuracy by Retrieval Depth (k)
   - Accuracy by Chunk Size
   - Accuracy by Chunk Overlap
   - Total Cost
