# JS Quickstart Examples

These mirror the Python quickstart topics, adjusted for the JS SDK:

1. `01_simple_qa.mjs` - basic local optimization
2. `02_customer_support_rag.mjs` - retrieval-style tuning
3. `03_custom_objectives.mjs` - explicit weighted / banded objectives through the JS spec

Run them after building the SDK:

```bash
npm run build:sdk
node examples/quickstart/01_simple_qa.mjs
```
