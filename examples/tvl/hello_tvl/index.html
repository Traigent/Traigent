Hello TVL

Beginner 5 min Core sections introduction

What You'll Learn

This minimal TVL spec introduces the core sections of the Tuner Variable Language, providing a foundation for defining optimization experiments.

  • Define a configuration_space with categorical, continuous, and discrete parameters
  • Specify objectives with maximize/minimize directions
  • Add basic constraints using CEL expressions
  • Set up promotion gates for deployment
  • Configure optimization strategy and budget

TVL Spec Structure

spec: id: hello-tvl version: 0.0.1 metadata: owner: "students@traigent.ai" description: "Minimal TVL spec" units: time: [milliseconds, seconds] ratio: [dimensionless] configuration_space: model: type: categorical values: ["gpt-4o-mini", "claude-3-haiku"] temperature: type: continuous range: [0.0, 0.8] unit: ratio objectives: - name: faithfulness direction: maximize - name: latency direction: minimize constraints: - id: mild-temperature type: expression rule: 'params.temperature <= 0.8' promotion: gate: manual_review reviewers: ["mentor@traigent.ai"]

Running This Example

# Validate the TVL spec tvl-lint examples/tvl/hello_tvl/hello_tvl.tvl.yml # Check structural validity tvl-check-structural examples/tvl/hello_tvl/hello_tvl.tvl.yml
