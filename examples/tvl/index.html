Tuned Variable Language (TVL)

TVL is a typed, analyzable configuration language for defining optimization experiments. It provides explicit constraints, units, objectives, and promotion gates for tuning pipelines with full validation and CI integration.

Key Features

  • Typed Configuration Space: Categorical, continuous, discrete, and integer parameter types with unit annotations
  • CEL Constraints: Expression, conditional, and forbidden constraint types with clear error messages
  • Multi-Objective Optimization: Pareto-optimal strategies with configurable budgets and timeouts
  • Environment Overlays: Extends/overrides pattern for staging and production configurations
  • Promotion Gates: Manual review, strict dominance, and epsilon-Pareto policies

TVL Tooling

  • tvl-lint - Validate TVL spec syntax and structure
  • tvl-check-structural - Check structural validity and constraint syntax
  • tvl-check-operational - Validate operational configuration with environment resolution
