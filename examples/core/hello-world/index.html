Hello World

● Beginner ⏱️ 5 minutes 📦 No API keys required

What You'll Learn

Build your first Traigent workflow by combining a direct sentiment prompt with optional retrieval augmentation. You'll see how Traigent evaluates responses, normalizes answers, and records results locally when running in mock mode.

Key Concepts

  • Sentiment prompt design
  • Retrieval-Augmented Generation toggles
  • Answer normalization
  • Local mock execution

Running This Example

TRAIGENT_MOCK_MODE=true python examples/core/hello-world/run.py

Mock mode saves runs to .traigent_local inside the example folder, so you can experiment without API keys. Switch TRAIGENT_MOCK_MODE off to call the configured provider using your own credentials.
