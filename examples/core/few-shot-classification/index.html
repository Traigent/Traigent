Few-Shot Classification

● Beginner ⏱️ 5 minutes 📦 No API keys required

What You'll Learn

In this example, you'll learn how to optimize example selection for few-shot classification tasks. Traigent will automatically find the best combination of examples to include in your prompts, improving classification accuracy without manual trial and error.

Key Concepts

  • Few-shot learning with LLMs
  • Example selection optimization
  • Classification accuracy measurement
  • Configuration space for few-shot parameters

Running This Example

TRAIGENT_MOCK_LLM=true python examples/core/few-shot-classification/run.py

This example runs in mock mode by default, so no API keys are needed. The optimization process will test different example selection strategies and identify the configuration that achieves the best classification accuracy.
