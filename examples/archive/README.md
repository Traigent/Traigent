# Examples Archive

Legacy artifacts, debug outputs, and superseded assets are stored here for reference. Nothing in this folder is executed by default or referenced from documentation.

## Structure

- `results/`: Historical JSON reports kept for auditing.
- `core/`: Backup prompts, logs, and experiment outputs migrated from the original numbered examples (e.g., `core/multi-objective-tradeoff`).

New artifacts should not be added here without a clear retention reason. Prefer updating the active examples instead.
